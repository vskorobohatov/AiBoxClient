# AiBoxClient

This project contains a simple Python client that sends an audio file to a local transcription API.

## Setup

Launch `install.sh` script in your terminal

## Start

Activate venv before the first start

```
source $VENV_DIR/bin/activate
```

Add m4a file to the files directory.

Start example:
```
python client.py ./files/[file_name].m4a
```
